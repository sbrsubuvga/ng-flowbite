# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-01-XX

### Added
- Initial release of ng-flowbite
- Modal component with NgfModalService
- Offcanvas component with NgfOffcanvasService
- Dropdown component
- Tooltip directive
- Accordion component
- Button component with multiple colors and sizes
- Card component with header, content, footer, and image support
- Badge component
- Alert component with dismissible support
- Toast service with success, error, warning, and info methods
- Demo application showcasing all components
- Unit tests for services and components
- Comprehensive documentation

### Features
- Service-based architecture similar to ng-bootstrap
- Full TypeScript support
- Tailwind CSS styling following Flowbite patterns
- Dark mode support
- Accessibility features (ARIA attributes)
- Responsive design
